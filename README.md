# Stock Price Analysis and Prediction

## Project Overview
This project focuses on real-world financial data analysis and stock price prediction using Python and machine learning techniques. The project analyzes historical stock market data to understand trends, visualize patterns, and predict future stock prices.

## Objective
The main objective of this project is to apply data science skills in the finance domain by performing:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Stock Trend Analysis
- Machine Learning Prediction

## Dataset
The dataset contains stock market information such as:
- Date
- Open Price
- High Price
- Low Price
- Close Price
- Trading Volume

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Features
- Analyzed stock market trends
- Created visualizations using graphs
- Generated correlation heatmaps
- Built a Linear Regression prediction model
- Evaluated model performance using metrics

## Machine Learning Model
A Linear Regression model was used to predict stock closing prices based on historical stock market data.

## Outcome
This project helped in understanding real-world financial data analysis and the application of machine learning techniques for predictive modeling.

## Conclusion
The project successfully demonstrated stock trend analysis, data visualization, and prediction using machine learning. It improved analytical thinking and practical data science skills.
